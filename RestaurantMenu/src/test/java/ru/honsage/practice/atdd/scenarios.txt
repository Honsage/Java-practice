Сценарий 1. Отображение блюд выбранной категории
Предусловие: Меню содержит блюда разных категорий (например, "Напитки")
Действия:
1. Пользователь выбирает определенную категорию (например, "Напитки")
2. Пользователь фильтрует меню по выбранной категории
Ожидаемый результат: Отобразится список всех блюд из категории "Напитки"

Сценарий 2. Обработка несуществующей категории
Предусловие: Меню не содержит блюд определенной категории (например, "Десерты")
Действия:
1. Пользователь фильтрует меню по определенной категории (например, "Десерты")
Ожидаемый результат: Отобразится пустой список блюд

Сценарий 3. Поиск блюда по части названия
Предусловие: Меню содержит определенное блюдо (например, "Овсяная каша")
Действия:
1. Пользователь определяет название (например, "Каша")
2. Пользователь вызывает поиск выбранного названия в меню
Ожидаемый результат: Отобразится блюдо "Овсяная каша"

Сценарий 4. Просмотр информации о блюде
Предусловие: В меню есть определенное блюдо и информация о нем (например, "Говяжий стейк", его цена и описание)
Действия:
1. Пользователь определяет блюдо (например, "Говяжий стейк")
2. Пользователь запрашивает информацию о выбранном блюде
Ожидаемый результат: Отобразится полная информация о "Говяжьем стейке": цена и описание
